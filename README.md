# Reproductor de Audios Musicales

Este proyecto es un reproductor web simple de audios musicales con una interfaz básica en HTML y CSS.

## Características
- Menú principal con botones para cada personaje o escena.
- Reproductor de audio incorporado para cada archivo MP3.
- Estilos simples para una experiencia visual clara.

## Estructura
- index.html: página principal.
- templates/: páginas individuales para cada audio.
- mp3/: archivos de audio.
- style.css: estilos generales.

## Uso
1. Abre index.html en tu navegador.
2. Selecciona el personaje o escena deseada.
3. Reproduce el audio desde el reproductor.

## GitHub Pages
El sitio está pensado para publicarse desde la rama principal del repositorio y usar rutas relativas para que funcione correctamente en GitHub Pages.
